Paper Mockup

Following the ideations, we were tasked to create a paper prototype of our favourite idea. At this point, I knew it was for the workbook, so I used the elements I liked most from all my ideas and settled for a desktop with a Twitch streamer interface. The main idea is that there is a character who is streaming the desktop screen, with a chat screen that has pre-generated messages from the viewers and an occasional donation that would show up. The character would be looping a gif and occasionally swap between different gifs to adjust to the situation. There would then be a file called Workbook.exe that can be opened to present a windows xp like window screen with the different weeks to be explored.
