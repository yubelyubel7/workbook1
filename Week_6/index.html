For week 6 we continued diving deeper into the world of p5.js and its capabilities. This week specifically though, we learnt more on the idea of data visualisation. How can we recreate boring old data into something more visual and appealing to the eye? We actually did a workshop activity revolving this, but that will be covered in another video!

As for the p5.js practise, this week we learnt more about implementing sound and music, in which it would be used to create visual effects for our text and/or images. In particular we used the p5.Amplitude function along with the volume and mappedVol in order to create a visual effect reacting to the sound file given.
