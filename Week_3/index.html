Media Queries

Media queries are used to allow websites to adapt their layout based on screen size:

  • Mobile (≤ 768px): Windows stack vertically by using auto for sizing, full width of images for better readability and preventing overflow
  • Tablet (769px - 1024px): Adjusted window positions, optimized for medium screens
  • Desktop (≥ 1025px): Full overlapping windows layout with original positioning

Responsive Design

Responsive web design, or RWD, is a design approach that addresses the range of devices and device sizes, enabling automatic adaption to the screen, whether the content is viewed on a tablet, phone, television, or watch. (wikipedia)

A good Ui/Ux-er has to be able to adapt their websites or apps for the different mediums they can be accessed.
